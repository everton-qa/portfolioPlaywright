Checkout – IA como Copiloto

A estrutura da pasta Checkout foi construída utilizando o MCP do próprio Playwright como copiloto.

O objetivo foi demonstrar:

Uso estratégico de IA para ganho de produtividade

Organização com Page Object Model

Estrutura escalável de testes

Capacidade de supervisionar e validar código gerado por IA

A IA foi utilizada como ferramenta. A arquitetura, entendimento e validações foram conduzidas por mim.
